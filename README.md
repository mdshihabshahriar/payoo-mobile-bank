# 💳 Payoo Mobile Bank

**Payoo Mobile Bank** is a simple mobile banking system built using **HTML, Tailwind CSS, DaisyUI, and JavaScript**.  
This project simulates the basic features of a mobile banking application where users can perform financial transactions through a clean and interactive interface.

---

## 🚀 Features

The system includes the following functionalities:

- 💰 Add Money to account balance
- 📤 Send Money to another user
- 🏧 Cash Out from account
- 🎁 Get Bonus balance
- 🧾 Pay Bills
- 📊 View Transaction History

These features demonstrate how a mobile banking interface can work using **frontend technologies and JavaScript logic**.

## 🛠️ Technologies Used

This project was built using:

- **HTML5** – Page structure  
- **Tailwind CSS** – Utility-first CSS framework  
- **DaisyUI** – Component library for Tailwind CSS  
- **JavaScript (Vanilla JS)** – Application logic and DOM manipulation

---

## 📂 Project Purpose

The goal of this project is to:

- Practice **Frontend Web Development**
- Learn **DOM Manipulation with JavaScript**
- Build a **Mobile Banking UI simulation**
- Implement **basic transaction logic**

This project is mainly created for **learning and demonstration purposes**.

---

## ⚠️ Limitations

- No backend server
- No database integration
- Data is not permanently stored
- All operations are handled using **JavaScript in the browser**
